# MB MARININI - ფეშნის მაღაზია

**MB MARININI** - არის თანამედროვე ონლაინ მაღაზია, რომელიც წარმოადგენს სტილიშ ტანსაცმელი, მოდა და ცხოვრების აქსესუარებს. ინსპირირებული სილამაზით და ენერგიით Waikiki-ის მაღაზიებისა, ჩვენ ქმნით ურთულეს შოპინგ გამოცდილებას.

## ფीচার

- React-ი ფუნქციური კომპონენტებით
- React Hooks: `useState`, `useEffect`, `useContext`, `useMemo`
- React Router ნავიგაციისთვის გვერდებს შორის
- Axios API ინტეგრაცია ტანსაცმელი ডाটასთან
- ლოკალური საცავი კალათისთვის და თემის შენახვა
- სესიის საცავი ბოლო დანახული გვერდისთვის
- ანიმირებული გეროვიბი და მოდალური ფანჯრები Framer Motion-ით
- მუქი/ღია თემის ართამობა
- მოწოდებული ენა: ქართული და ინგლისური
- რეაქტიული დიზაინი მობილური, ტაბლეტ და დესკტოპის მოწყობილობებისთვის
- SASS/SCSS სტილი და კომპონენტის ბაზაზე აგებული

- ჩვენს შესახებ
- გვერდი ვერ მოიძებნა

## ლოკალურ შენახვაზე გაშვება

1. დამოკიდებელობების ინსტალაცია

```bash
npm install
```

2. დეველოპმენტის სერვერის დაწყება

```bash
npm run dev
```

3. პროდუქციის ბილდი

```bash
npm run build
```

## პროექტის სტრუქტურა

- `src/App.jsx` — მთავარი აპლიკაცია და რუტის კონფიგ
- `src/main.jsx` — React შესვლის წერტილი
- `src/context` — თემა, ენა და კალათის კონტექსტი
- `src/hooks` — კუსტომ საცავის hooks
- `src/services/api.js` — Axios API კლიენტი
- `src/components` — განლაგება, კარტები, მოდალები
- `src/pages` — გვერდი ხედები და SCSS სტილი
- `src/styles` — გლობალური სტილის შესვლის წერტილი

## განვითარება

ეს აპლიკაცია მზად არის განვითარებისთვის Netlify-ზე, GitHub Pages-ზე, Vercel-ზე ან ნებისმიერი სტატიკური ჰოსტინგ სერვისზე, რომელიც Vite-ს მხარს უჭერს.
